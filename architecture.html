Lab Architecture

The Hybrid DevOps Lab is composed of interconnected local and cloud components designed to simulate a secure, production-grade infrastructure. The CI/CD layer is built around GitHub Actions and Jenkins. The GitOps layer is powered by KIND and ArgoCD. Security is enforced using Vault (with OIDC), SOPS, and strict ClusterIP service exposure via NGINX ingress.
