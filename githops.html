GitOps Workflow

GitOps provides the foundation for automated and auditable Kubernetes deployment in the lab. ArgoCD watches for changes in Git repositories and continuously synchronizes desired states to the cluster.

Core Concepts

  • ArgoCD: Monitors Helm chart repos and applies updates to Kubernetes automatically.
  • Multi-Environment: Separate Helm value files and overlays for dev, staging, and prod.
  • CI Integration: Jenkins commits chart updates post-build, triggering ArgoCD sync.
  • Rollouts: Canary or Blue/Green deployments are controlled via Argo Rollouts CRDs.

Key Benefits

  • Git as the single source of truth
  • Rollback with Git commit history
  • Drift detection & reconciliation
  • Full observability into rollout status
