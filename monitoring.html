Monitoring & Observability

The lab implements full observability using Prometheus, Grafana, and Alertmanager. It captures application metrics, infrastructure state, and security alerts.

Monitoring Stack

  • Prometheus: Scrapes metrics from app services, ArgoCD, Jenkins, Vault, and Kubernetes API.
  • Grafana: Dashboards visualize deployments, pod metrics, CI pipeline runs, and secrets audit.
  • Alertmanager: Routes notifications to Slack via webhook, based on severity and rule groups.
  • Node Exporter: Collects EC2 host metrics (CPU, RAM, Disk, etc.)
  • Blackbox Exporter: Optional probe for uptime and HTTPS validation.

Examples

  • Jenkins Job Failure → Slack Alert
  • High Pod Memory Usage → Grafana Dashboard Warning
  • Vault Policy Violation → Prometheus Rule → Alertmanager → Slack
