Prerequisites

  • Docker Desktop
  • Git + GitHub account
  • Vault CLI and Docker image
  • kubectl + kind
  • Helm v3+

Project Phases

  1. Phase 1: Initialize Maven App, Jenkins, SonarQube, MinIO, Prometheus, Grafana
  2. Phase 2: Add ArgoCD, Vault Agent Injector, Ingress, KIND overlay
  3. Phase 3: Prebuilt Microservices + Docker Hub + Secure CI with Vault OIDC
  4. Phase 4: GitOps multi-env, Rollouts, Alertmanager, Slack integration

Setup Instructions

  • Clone repo and start with setup.sh
  • Use vault auth enable oidc to configure secure access
  • Push image updates to trigger GitHub Actions → Jenkins → ArgoCD
  • Use kubectl port-forward svc/grafana for UI testing
