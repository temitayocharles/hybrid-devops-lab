Welcome to the Hybrid DevOps Lab

This lab simulates a fully integrated, secure, and production-ready DevOps environment designed to showcase real-world best practices. The project uses a hybrid model—blending local infrastructure (Docker, Jenkins, Vault, MinIO) with Kubernetes-based GitOps delivery, CI/CD automation, secrets management, image scanning, and alert routing.

Explore how GitHub Actions triggers Jenkins to deploy apps via ArgoCD. Discover the role of Vault OIDC for secure identity management, and learn how a strict ClusterIP + Ingress strategy secures all services behind a single proxy.

The site provides full documentation for each system component and implementation stage. Use the navigation bar above to explore CI/CD pipelines, GitOps patterns, secrets handling, observability dashboards, and much more.
